== Installation ==

1. Unzip all files to the `wp-content/plugins/hrpress-lite` directory
2. Log into WordPress admin and activate the 'HrPress Lite' plugin through the 'Plugins' menu
3. Go to *Hr Press Lite > Dashboard* in the left-hand menu to start using


More information on using the plugin is [available here](https://codeclove.com/plugins/hr-press).
